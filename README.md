# We-promise Tap

## How do I install these formulae?

`brew install we-promise/tap/<formula>`

Or `brew tap we-promise/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "we-promise/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
